# lunarvim-config
lunarvi-config

Put this lvim folder under "$HOME/.config" folder. If there is already a lvim folder, remember to make a backup of that existing one first.

```
lvim
├── config.lua
└── lua
    └── user
        ├── autocmds.lua
        ├── banners.lua
        ├── colorscheme.lua
        ├── dashboard.lua
        ├── filetype.lua
        ├── keymaps.lua
        ├── kind.lua
        ├── lsp.lua
        ├── metals.lua
        ├── options.lua
        ├── plugins.lua
        ├── statusline.lua
        └── which-key.lua
```
